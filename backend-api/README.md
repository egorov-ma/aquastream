# API Gateway

API Gateway на базе Spring Cloud Gateway для маршрутизации запросов к микросервисам.

## Функциональность

- Маршрутизация запросов
- Балансировка нагрузки
- Аутентификация и авторизация
- Rate limiting
- Circuit breaking
